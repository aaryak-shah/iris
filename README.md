> See an [example here](./example/example_api.dart)

# Local development setup
* Clone this repository onto your local system
* Install dependencies
```sh
dart pub get
```

* Set up pre-commit
```sh
chmod +x setup_git_hooks.sh
./setup_git_hooks.sh
```
